# 推し活メモ

推し活・オタ活に特化した支出管理アプリです。
メンバーカラーに合わせたテーマカスタマイズ、月別の支出管理、年間合計表示、二重登録防止機能を搭載しています。

## 機能

- 💖 支出の記録（カテゴリ・日付・内容・金額・メモ）
- 📋 支出一覧表示（日付グループ）
- 📅 月切り替え（前月・翌月・今日へ戻る）
- 💰 選択月の合計表示
- 🗓️ 選択年の年間合計表示
- 🎨 メンバーカラー変更（プリセット5色 + カスタムカラー）
- 🛡️ 二重登録防止（ハッシュ照合・領収書番号照合・連打防止）

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フレームワーク | Next.js 14（App Router） |
| 言語 | TypeScript 5 |
| CSS | Tailwind CSS 3 |
| データ保存 | localStorage |
| 状態管理 | React Context API |
| フォント | Noto Sans JP |

外部UIライブラリは一切使用していません。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ディレクトリ構成

```
src/
├── app/             # Next.js App Router
├── components/      # UIコンポーネント
├── config/          # 定数・カラー設定・storageラッパー
├── context/         # ThemeContext（メンカラ管理）
├── hooks/           # useExpenses（支出データCRUD・月/年集計）
└── types/           # TypeScript型定義
```

## 重複登録防止の仕組み

| レベル | 内容 |
|---|---|
| ハッシュ照合 | 日付・金額・カテゴリ・内容の完全一致を検知 |
| 領収書番号照合 | 注文番号・発券番号の重複をシャットアウト |
| 連打防止 | 送信ボタンを即時 disabled 化 |
